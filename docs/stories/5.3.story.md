### Changelog Format

Follow [Keep a Changelog](https://keepachangelog.com/) + [Semantic Versioning](https://semver.org/):

```markdown
# Changelog

## [1.0.0] - 2026-XX-XX

### Added
- Offline OCI registry daemon (buncker)
- Online fetch CLI (buncker-fetch)
- AES-256-GCM encrypted USB transfers
- BIP-39 mnemonic pairing
- Delta sync (only missing blobs)
- Dockerfile analysis and resolution
- Manual garbage collection
- .deb packaging for Debian/Ubuntu
- systemd service with security hardening
```
